# Himaaweb - Sistem Informasi Himpunan Mahasiswa

[![Project Status](https://img.shields.io/badge/status-development-orange.svg)](https://github.com/rubuproject/himaaweb)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Himaaweb** adalah platform berbasis web yang dirancang khusus untuk mengelola aktivitas, data anggota, dan publikasi informasi bagi Himpunan Mahasiswa (Hima). Proyek ini bertujuan untuk mendigitalisasi proses administrasi agar lebih efisien dan transparan.
